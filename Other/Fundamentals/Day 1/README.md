#Day 1 

##input

    integer N 
    0 >= N > 1000    
    
##output

    The number entered is <N>
    The sum from 1 to <N> is <1 + 2 + 3 + ... N> 
    There are <X> even numbers between 0 and N (including N)
        
##example

input

    6
    
output
    
    Then number entered is 6
    The sum from 1 to 6 is 21
    There are 4 even numbers between 0 and 6
