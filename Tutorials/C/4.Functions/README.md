#Function help organize the code and reduce repetition 

function syntax

    (return type) (name)()
    {
        <code>
        return type
    }
    

without function 

    int main()
    {
        <lots of code>
        <will compile>
    }
    
with functions
    
    int main()
    {
        <code>
        function_name() <calling function> 
    }
    
    int function_name()
    {
        <code>
        <return> 
    }
